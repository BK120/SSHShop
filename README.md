<h1>基于SSH框架的网上商城 </h1>
本项目开发环境为：eclipse4.6+tomcat9+oracle10g+win7     
项目中ssh相关框架版本为：spring3.2+struts2.0+hibernate3.0
项目中可吸收的部分知识点为：c3p0数据库连接池、图形化验证码输出、MailServer服务器邮箱验证、在线支付易宝支付以及后端界面实现Dtree框架
 JQuery、ajax异步验证 Struts2后端验证界面回显
